**Installation**
----------------

    npm install -g react-native-run-helper

**Usage**
---------

Simple run on android

    rnrun
Or
    rnrun [option]

Run with clean project

    rnrun clean

Assemble Release APK (It Will Clean It too, so no need for clean here)

    rnrun build-release

Open Apks Dir

    rnrun show-apks

To Prevent run with any option

    rnrun [option] not-run
